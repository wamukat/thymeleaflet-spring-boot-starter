Thymeleaflet

Thymeleaflet は Thymeleaf フラグメント向けの Storybook 風 UI です。 フラグメントを検出してプレビューを表示し、HTML テンプレート内の JavaDoc 風コメントから使用例を生成します。

Live demo → GitHub repository →
はじめに → Stories ガイド → 設定 →

注: Thymeleaflet は開発環境・社内環境向けの UI です。

概要

Thymeleaflet は Thymeleaf フラグメントを素早く確実に確認するためのプレビュー UI です。 ストーリーとライブプレビューで実装の意図を常に見える状態にします。

ストーリー、プレビュー、フラグメント詳細をまとめて確認。

ストーリー作成 (stories.yml)

stories.yml で UI のバリエーションを定義し、期待する状態を明示的に管理します。 

storyGroups:
  accordionItem:
    title: Accordion Item
    description: Vanilla JS accordion using inline handlers
    stories:
      - name: default
        title: Closed
        parameters:
          title: Shipping details
          body: Ships in 2-3 business days.
          open: false
      - name: open
        title: Open
        parameters:
          title: Returns
          body: Returns are accepted within 14 days of delivery.
          open: true
実際の stories.yml での定義例。
UI に反映されたストーリー一覧。

ライブプレビュー

ストーリーを選ぶとプレビューが即時更新されます。パラメータやモデルの編集も同じ画面で完結します。

ストーリー選択と同時にプレビューが更新。

フラグメント詳細 (JavaDoc)

JavaDoc 風コメントからフラグメントの説明やモデル仕様を抽出し、意図を見える化します。 

/**
 * Select input (selectInput)

はじめに

スターターをビルドせずに利用できます。依存を追加すればすぐに試せます。

依存関係

Maven

<dependency>
  <groupId>io.github.wamukat</groupId>
  <artifactId>thymeleaflet-spring-boot-starter</artifactId>
  <version>x.y.z</version>
</dependency>

Gradle (Groovy)

implementation 'io.github.wamukat:thymeleaflet-spring-boot-starter:x.y.z'

設定の概要

application.yml でプレビューのビューポートや背景色を設定できます。 

thymeleaflet:
  preview:
    backgroundLight: "#f6f7fb"
    backgroundDark: "#1f2331"
    viewports:
      - id: mobileSmall
        labelKey: thymeleaflet.preview.viewport.mobileSmall
        width: 360
        height: 640
      - id: tablet
        labelKey: thymeleaflet.preview.viewport.tablet
        width: 834
        height: 1194
